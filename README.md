# Este es el juego de Número Secreto - Alura Latam

La idea es adivinar un número entre 1 y un número asignado según el juego (actualmente hasta 30). 

La página [gitignore.io](https://www.toptal.com/developers/gitignore/) sirve para identificar los archivos/carpetas que no deben estar de manera pública en el repositorio remoto.
